## 常见数据结构
 - 数组 Array
 - 栈 Stack
 - 队列 Queue
 - 链表 Linked List
 - 树 Tree
 - 哈希表 Hash
 - 堆 Heap
 - 图 Graph
 
 | 结构 | 优点  | 缺点 |
 | ----| -----| -----|
 | 数组 |插入快 |查找和删除慢，大小固定|
 | 栈 | 后进先出|  非优|
 | 队列| 先进先出 | 非优 |
 | 链表| 插入快，删除快 | 查找慢| 
 | 二叉树| 平衡树，查找、插入、删除快| 算法复杂 | 
 | 哈希表 | 已知key,存取快 | | 
 |堆| 插入、删除快
